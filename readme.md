# img2ascii

A CLI tool that maps image pixels to ASCII characters.

Built with Python, once you have Python installed, you're good to run it. :)

## How to use
`./img2ascii -i input_image.jpg`

or

`./img2ascii -i input_image.jpg -sc 0.2`
